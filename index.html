Projects and languages


The Processing Foundation
Processing is a flexible software sketchbook and a Java-based language for learning how to code within the context of the visual arts. p5.js is based on the same metaphor, but allows you to create for the web, using JavaScript, HTML, and CSS. Processing.py is also similar to Processing, but uses the Python language.
It involves the following languages:

  • JavaScript

  • Java

  • Python


Styled Components
styled-components is a new, fabulous way to style React.js applications. (+ other frameworks in the future) Utilising the full power of ES6, the newest version of JavaScript, and CSS it has a bunch of interesting properties that make it unlike anything you've seen before.

  • JavaScript

  • CSS

  • React
